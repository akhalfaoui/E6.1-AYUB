Atributo Descripcion Explicacion Ejemplo
color Canvia el color de las letras del selector. como parametro se pone el color en formato hex,rgb o rgba. color: red;
color: #FF0000;
color: rgba(255,0,0);
background-color Cambia el color de fondo del selector. Como parametro se pone el color en hex, rgb, rgba o nombre del color. color: #2DC751;
background-image Sirve para agregar diversas imagenes com una sola . Sus parametro es url('...').

background-image: url("ejemplo1.gif")
background-repeat Sirve para repetir la imagen que elijas tanto horizontalmente como verticalmente, trabaja juntamente con background-image. Su parametros son repeat, repeat-x, repeta-y o no-repeat.

background-repeat: repeat-x;
font-size Sirve para aumentar o disminuir el tamaño al texto. Sus parametros son medium, small, large, smaller, larger, % o length. font-size: 80%;
font-weight Sirve para definir el grueso del texto. Sus parametros son normal, bold , lighter o number. font-weight: 800;
font-family Sirve para definir la fuente al elemento. Los parametros son nombres de fuentes. font-family: Arial;
font-align Sirve para definir la posicion del texto. Sus parametros son left, right, center o justify. text-align: center;
text-tranform Sirve para cambiar el texto en masuculas o minusculas segun el parametro que le digas. Sus parametros son none, capitalize, uppercase o lowercase. text-tranform: uppercase;
border Se utiliza para modificar los bordes de un elemento. Sus parametros son Style, color, width, top, left, right o bottom. border-color: blue;
list-style-type Se utiliza para listar el contenido con algun estilo. Sus parametros son disc, armenian, circle, georgian,lower, none, uppero decimal.
  • list-style-type: circle;
    		•
    text-decoration Sirver para añadir decoraciones al texto. Sus parametros son none, underline,overline o line-through. text-decoration: line-through;
    padding Sirve para definir el espacio. Sus parametros son bottom, left, top o right. Padding-left: 50px;
    margin Es otra opcion de definir espacio. Sus parametros son top, right, bottom o left.

    margin-left: 10px;
